XML and Javascript Information

What is XML?

XML, which stands for eXtensive Markup Language, is a language that was created to carry and store information. Unlike HTML, it will not display information. Separate programming must still be used.


How is XML used?

XML is used with tags that are made up by the document author, as opposed to HTML which has its own set of predefined tags. Adding, removing, or modifying data is simple in XML, too.

What is JavaScript?

JavaScript is a language that is interpreted by a web browser that utilizes and manipulates objects. This can include the browser's window, the actual document, or elements such as pictures.


How is JavaScript related to HTML and CSS?

JavaScript is related to HTML and CSS in a couple ways. First, Java scripting can be done directly in an HTML or file, or in a separate file like CSS would be. JavaScript also directly manipulates HTML elements such as buttons, forms, windows, images, etc.


Common JavaScript...

  • Elements: forms & images
  • Properties: properties are created values for an object. For example, object.property
  • Methods: methods for JavaScript would be written like 
    name = person.firstName()
  • Functions: Functions are activated when they're invoked, or called. The code to be executed are inside {. It would look like 
    function name (parameters) {}
